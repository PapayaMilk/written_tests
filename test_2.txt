注册接口：
请求参数
手机号 phonenum，密码 password 验证码 img_code

响应参数
用户名 username 接口状态码 code 接口信息 msg

接口逻辑：
获取前端发来的请求参数，验证手机号、密码、验证码是否符合规范，
验证手机验证码是否正确，查询数据库此手机号是否已注册过，若已注册
返回已注册提示；没有注册过，通过手机号，使用算法生成随机数字用户名，
将密码进行加密，然后将手机号、用户名、加密密码存入数据库中，
返回用户名、注册成功信息。

登录接口：
请求参数
手机号，密码 password

响应参数
用户名 username  token信息 token 接口状态码 code 接口信息 msg

接口逻辑
获取请求参数，查询手机号是否存在，不存在，返回用户未注册信息；
若手机号验证存在，将密码通过相同的加密方式进行加密，通过对比数据库中的密码，
验证密码是否正确，若密码正确，生成token，将用户名、token、登录成功信息返回。
